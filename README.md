# JARVIS_Voice_Assistant

A fully functional AI-powered voice assistant for Windows built using Python.
It performs tasks like speech recognition, text-to-speech, searching Wikipedia, opening applications, telling jokes, sending emails, browsing the web, and more**.

**Features**

**The assistant can:**

Listen using your microphone

Recognize speech using Google Speech Recognition

Speak responses aloud using pyttsx3

Internet Tasks

Search Wikipedia

Open YouTube

Open Google

Open StackOverflow

Play videos and perform searches

System Utility Tasks

Open Notepad

Open Calculator

Open Camera

Play music from your local folder

Tell jokes

Tell the current time

Email Sending (Customizable)

Uses Gmail SMTP to send emails (you can add your credentials).
